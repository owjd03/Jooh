# Jooh
we are team Joohfeed for lifehack 2025

Usage:

Windows
```
python -m venv .venv
.venv\Scripts\activate

pip install -r requirements.txt

python app.py
```

Mac
```
python -m venv .venv
source .venv/bin/activate

pip install -r requirements.txt

python app.py
```


add `GEMINI_API_KEY="<YOUR_API_KEY>"` in .env file in the root directory.
